# Yearbook

A simple HTML-based yearbook page for the Class of 2026.

## Overview

- Displays a grid of student cards with initials, names, and placeholder quotes.
- Includes a `Scramble` button to randomize the order of displayed students.
- Opens a profile modal when a student card is clicked.

## Files

- `main.html` — the full yearbook page with styling and interactivity.

## Usage

1. Open `main.html` in a browser.
2. Use the search input to filter students by name or quote.
3. Click `Scramble` to shuffle the student cards.
4. Click any student card to view their profile modal.

## Notes

- This is a static project and does not require a web server.
- The student data is stored directly inside `main.html`.
